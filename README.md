# 🛍️ Customer Shopping Behavior Analysis
End-to-end data analysis project using Python (Pandas), SQL and Power BI to analyze customer shopping behavior, generate business insights and build interactive dashboards.

## 📌 Project Overview

This project analyzes customer shopping behavior using transactional data to uncover insights into spending patterns, product preferences, and customer segmentation. The analysis helps support data-driven business decisions.

---

## 📊 Dataset

* **File:** `customer_shopping_behavior.csv`
* Contains customer demographics, purchase details, and shopping behavior.

### Key Features:

* Age, Gender, Location
* Item Purchased, Category, Purchase Amount
* Review Rating, Shipping Type
* Discount Applied, Subscription Status

---

## 🐍 Python Analysis

* **File:** `Customer_Shopping_Behavior_Analysis.ipynb`
* Tool: Pandas in Jupyter Notebook

### Steps performed:

* Data loading and exploration
* Handling missing values
* Data cleaning and preprocessing
* Feature engineering (age groups, purchase patterns)
* Summary statistics and insights

---

## 🗄️ SQL Analysis

* **File:** `Customer Shopping Behavior.sql`

### Key business queries:

* Revenue by gender
* Top 5 products by rating
* Customer segmentation (New, Returning, Loyal)
* Subscription vs non-subscription analysis
* Discount usage impact
* Revenue by age group

---

## 📈 Power BI Dashboard

* **File:** `Customer Behavior Dashboard.pbix`

### Dashboard includes:

* Sales overview
* Customer segmentation
* Product performance insights
* Interactive filters for analysis

---

## 🛠️ Technologies Used

* Python (Pandas)
* SQL (MySQL / PostgreSQL)
* Power BI
* Jupyter Notebook
